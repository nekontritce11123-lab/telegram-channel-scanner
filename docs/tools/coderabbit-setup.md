# CodeRabbit Setup Guide

CodeRabbit is a GitHub App that provides AI-powered code reviews for pull requests.

## Setup Instructions

1. **Visit** https://app.coderabbit.ai/login
2. **Login** with your GitHub account
3. **Add repositories** you want CodeRabbit to review
4. **Done** - CodeRabbit will automatically review all new PRs

## Features

- Automatic PR code review
- AI-powered suggestions and improvements
- Security vulnerability detection
- Code quality analysis

## Notes

- CodeRabbit is a GitHub App, not a CLI tool
- Reviews are triggered automatically on new PRs
- No local installation required
